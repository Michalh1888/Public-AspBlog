Pro spuštění projektu:
*je potřeba znovu vytvořit databázi(protože je to na localhostu ve složce usera)
*Postup:
-otevřít projekt=>spustit Package Manager Console(View=>další okna)=> příkaz PM: Update-Database
-znovu spustit projekt a zaregistrovat admina emailem: admin@admin.cz(heslo libovolné)
-odhlásit se a znovu spustit projekt
-přihlásit se jako admin
Pozn.: admin může vytvářet a editovat články, user pouze číst(zaregistr.libovol.emailem=>user)
